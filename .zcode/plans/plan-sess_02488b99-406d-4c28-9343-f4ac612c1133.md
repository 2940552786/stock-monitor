## 布局调整 + 筹码图纵向化方案

### 目标效果
将筹码分布图从分时图下方移到右侧信息栏，纵轴改为价格（与分时图价格轴对齐），横轴为筹码占比百分比。布局变为：

```
┌──────────┬──────────────────────┬──────────────┐
│ 自选股    │   分时图 (flex:1)     │  股票详情     │
│ (210px)  │                      │  - 价格/涨跌  │
│          │                      │  - OHLCV     │
│          ├──────────────────────│  - 顶底信号   │
│          │   信号监控台 (flex:1)  ├──────────────│
│          │                      │  筹码分布图   │
│          │                      │  纵:价格↑     │
│          │                      │  横:占比%→    │
│          │                      │  指标数据     │
└──────────┴──────────────────────┴──────────────┘
```

### 具体改动

#### 1. HTML 结构调整
- `#chip-chart` 从 `#center-panel` 移到 `#info-panel` 内部（放在信号区域下方、筹码指标上方）
- `#info-panel` 改为 flex column 布局，股票详情在上，筹码图在中，指标在下

#### 2. CSS 调整
- `#trend-chart`：`height: 52%` → `flex: 1`（与信号监控台平分中心区域）
- `#chip-chart`：移除旧的 `flex-shrink: 0` 固定高度，改为在右侧面板内 `height: 200px; flex-shrink: 0`
- `#info-panel`：添加 `display:flex; flex-direction:column;` 确保内容按序排列

#### 3. 分时图 (updateTrendChart)
- 保持现有逻辑（价格线、均价线、顶底标记）
- **新增**：显式计算 Y 轴范围（`min`/`max`），并存入 `STATE.trendYRange`，供筹码图对齐
- 移除旧的 `markLine` 昨收线（保留）

#### 4. 筹码图 (updateChipChart) 重写
- **Y 轴 (纵)**：`type: 'value'`，价格，`min`/`max` 与分时图一致（从 `STATE.trendYRange` 读取），`position: 'right'`，`inverse: true`（价格从高到低，与分时图方向一致）
- **X 轴 (横)**：`type: 'value'`，筹码占比百分比
- **系列**：使用 `line` + `areaStyle` 绘制横向筹码分布曲线
  - 获利盘区域（红色半透明）：价格 ≤ 现价的部分
  - 套牢盘区域（绿色半透明）：价格 > 现价的部分
- **标记线 (markLine)**：
  - `yAxis` 位置标当前价（蓝色实线）
  - `yAxis` 位置标平均成本（金色虚线）
- 整体紧凑设计，适配 ~200px 高度

#### 5. JavaScript 其他调整
- `initCharts()`：保持不变，`chipChart` 自动绑定到新位置的 DOM 元素
- `fetchAllData()`：先更新趋势图（计算 Y 范围），再更新筹码图（使用该范围）
- 窗口 resize 逻辑不变

### 不修改的部分
- 自选股面板
- 信号监控台
- 后端 server.py
- 筹码指标数值显示（`updateChipInfo`）
